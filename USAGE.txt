fixnix - a nix expression fixer (version 0.0.0)

Usage: fixnix COMMAND

Available options:
  -o,--fix-folder FOLDER   The relative path to the fix folder.
                           (default: "nix/fix/")
  -f,--force               Don't ask anoying questions, just do!
  -h,--help                Show this help text

Available commands:
  print                    print the fix-file to stdout
  add                      add/override fix-file(s) to the fix-folder
  list                     list the locations and their formats
